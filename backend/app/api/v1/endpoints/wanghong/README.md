# 🇨🇳 왕홍(Wanghong) 데이터 분석 및 선별 가이드

이 프로젝트는 하이툰(Huitun)에서 수집된 99명의 가구/인테리어 분야 왕홍 데이터를 바탕으로, 우리의 비즈니스 목적에 가장 적합한 인플루언서를 DeepSeek을 통해 랭킹화하는 과정을 담고 있습니다.

## 📋 데이터 선별 프로세스
1. **데이터 수집**: `2-data.py`를 통해 99명의 기초 데이터 수집 (`influencers.json`)
2. **AI 분석**: `influencers.json`의 내용을 아래 프롬프트와 함께 DeepSeek에 전달
3. **최종 랭킹**: AI가 판별한 연관성 점수에 따라 상위 리스트 추출

---

## 🤖 DeepSeek 최적화 프롬프트 (복사해서 사용하세요)

**[지시사항]**
제공된 99명의 왕홍 데이터(`json`)를 바탕으로, 우리가 설정한 **[캠페인 목적]**과 가장 연관성이 높은 순서대로 상위 **[원하는 인원수, 예: 10명]**의 랭킹을 매겨주세요.

**[캠페인 목적/상세 내용]**
> "여기에 우리가 입력했던 상세 내용(예: 모던한 가구 홍보, 신혼부부 타겟, 가성비 위주 등)을 입력하세요"

**[AI 분석 및 연관성 평가 기준]**
DeepSeek은 아래 4가지 기준에 가중치를 두어 `Relativity Score (0-100)`를 산출하고 랭킹을 정해주세요:
1. **콘텐츠 정합성 (40%)**: 왕홍의 활동 설명과 이름이 우리 가구/인테리어 컨셉과 얼마나 일치하는가?
2. **성장 잠재력 (30%)**: 데이터의 `growth_rate`와 `growth_amount`를 분석하여 현재 뜨고 있는 계정인가?
3. **영향력 품질 (20%)**: `followers` 대비 `score`(추천지수)가 효율적으로 관리되고 있는가?
4. **타겟 적중도 (10%)**: 닉네임이나 설명에서 나타나는 주요 시청자 층이 우리 제품과 맞는가?

**[출력 형식]**
최종 결과는 반드시 아래 표 형식을 포함한 리스트로 제공하고, 마지막에는 **"연관성 판단 근거 요약"**을 작성해 주세요.
- 순위 | 이름 | ID | 연관성 점수 | 선별 이유 요약
